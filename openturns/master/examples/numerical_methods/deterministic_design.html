Deterministic design of experimentsÂ¶

In this example we are going to expose the available deterministic design of experiments that allow for example to determine the variation range of an output variable of interest.

Four types of deterministic design of experiments are available:

  • Axial

  • Factorial

  • Composite

  • Box

Each type of deterministic design is discretized differently according to a number of levels.

The generated samples lies in the unit cube and can be scaled then translated to cover the desired interval.

[1]:

from __future__ import print_function
import openturns as ot
import math as m

[2]:

# Axial design
levels = [1.0, 1.5, 3.0]
experiment = ot.Axial(2, levels)
sample = experiment.generate()
ot.Cloud(sample)

[2]:

[3]:

# scale & translate to get desired location
sample *= 2.0
sample += [5.0, 8.0]
ot.Cloud(sample)
