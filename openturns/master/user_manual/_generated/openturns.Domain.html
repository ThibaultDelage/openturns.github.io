Domain¶

class Domain(*args)¶

Domain.

Available constructors:
Domain(lowerBound, upperBound)
Parameters:

lowerBound, upperBound : sequence of float of dimension dim

Define a finite interval . It is allowed to have for some : it simply defines an empty interval. By default, an empty interval is created.

Notes

A Domain object can be created through its derived classes:

  • Mesh
  • RegularGrid
  • Interval
  • LevelSet

Examples

>>> import openturns as ot
>>> # Create the interval [a, b]
>>> a = 1
>>> b = 3
>>> print(ot.Domain([a], [b]))
[1, 3]
>>> print(ot.Domain(ot.Interval(a, b)))
[1, 3]
