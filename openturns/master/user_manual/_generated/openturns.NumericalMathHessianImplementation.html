NumericalMathHessianImplementation¶

class NumericalMathHessianImplementation(*args)¶

Numerical math hessian implementation.

See also

NumericalMathFunction, NumericalMathGradientImplementation

Notes

This object is the result of the method getHessian of a NumericalMathFunction.
