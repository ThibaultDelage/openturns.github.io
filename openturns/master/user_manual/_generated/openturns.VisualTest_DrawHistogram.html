VisualTest_DrawHistogram¶

(Source code, png, hires.png, pdf)

VisualTest_DrawHistogram(*args)¶

Draw an histogram.

Available usages:

VisualTest.DrawHistogram(sample)

VisualTest.DrawHistogram(sample, barsNumber)

Parameters
sample2-d sequence of float

Sample to draw.

barsNumberpositive int

Number of barplots used to draw the histogram. If not specified, it is automatically determined according to the Silverman rule.

Returns
graphclass:~openturns.Graph

The graph object

Examples

>>> import openturns as ot
>>> from openturns.viewer import View

Generate a random sample from a Normal distribution:

>>> ot.RandomGenerator.SetSeed(0)
>>> sample = ot.Normal(1).getSample(100)

Draw an histogram:

>>> hist = ot.VisualTest.DrawHistogram(sample)
>>> View(hist).show()
