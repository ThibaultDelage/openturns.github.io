Sampler¶

class Sampler(*args)¶

Sampler.

Available constructors:
Sampler(aSampler)
Parameters:

aSampler : Sampler

Particular sampler. By default it is a RandomWalkMetropolisHastings.

See also

MCMC, RandomWalkMetropolisHastings

Notes

A Sampler is an object whose fundamental ability is to produce samples according to a certain distribution.

Examples

>>> import openturns as ot
>>> sampler = ot.Sampler()
