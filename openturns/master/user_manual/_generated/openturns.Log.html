static Show(flags)¶

Accessor to the log flags.

Parameters:
flags : int

Log flags to enable.

Available values to be ORed:

  • DBG
  • INFO
  • USER
  • WARN
  • ERROR
  • TRACE
  • DEFAULT
  • NONE
  • ALL

Examples

>>> import openturns as ot
>>> ot.Log.Show(ot.Log.DEFAULT)
