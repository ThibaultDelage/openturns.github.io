Optimization¶

Generic optimization classes¶

OptimizationProblem(*args) Base class to define an optimization problem.
OptimizationAlgorithm(*args) Base class for optimization wrappers.
OptimizationResult(*args) Optimization result.

Optimization solvers¶

AbdoRackwitz(*args) Abdo-Rackwitz solver.
Cobyla(*args) Constrained Optimization BY Linear Approximations solver.
SQP(*args) Sequential Quadratic Programming solver.
TNC(*args) Truncated Newton Constrained solver.
NLopt(*args) Interface to NLopt.
MultiStart(*args) Multi start optimization algorithm.
EfficientGlobalOptimization(*args) Efficient Global Optimization algorithm.

Nearest point assessment¶

NearestPointChecker(*args) Proxy of C++ OT::NearestPointChecker.
NearestPointCheckerResult(*args) Proxy of C++ OT::NearestPointCheckerResult.
