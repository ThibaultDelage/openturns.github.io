Implements a shared pointer strategy

See Also
SharedPointer
InterfaceObject
Template Parameters
TThe class to point to
IMPLThe class used to actually implement the shared pointer

Pointer implements a shared pointer strategy. The pointee object is shared among all pointers so none has pointee ownership. When one of them wants to modify the object, it MUST deep-copy it (copy-on-write strategy).
