Detailed Description

The class RegularGrid implements an equaly spaced set of real values.

The RegularGrid class defines some regular time slice defined by three values:

  • either a start time, an interval (aka timeStep) and a number of steps
  • or a start time, a number of steps and an end time

Note: the end time conforms to the STL standard of "one past the end", so the following formula defines the end time: end time = start time + ( timeStep * steps )
