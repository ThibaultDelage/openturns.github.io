LinearModelAlgorithm¶

class otlm.LinearModelAlgorithm(*args)¶

Class used to create a linear model from numerical samples.

Available usages:

LinearModelAlgorithm(Xsample, Ysample)

See also

LinearModel, LinearModelResult
